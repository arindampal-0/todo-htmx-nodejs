# Todo App HTMX Nodejs

## Setup

```shell
pnpm install
pnpm build
pnpm start
```

OR

```shell
npm install
npm run build
npm start
```

### Dev Setup

```shell
pnpm install
pnpm dev
```

OR

```shell
npm install
npm run dev
```

## Features

-   CRUD functionality
-   Search Todo
-   Sort by
